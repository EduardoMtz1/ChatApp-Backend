# ChatApp - Backend

Chat application based on WebSockets and HTTP requests

## Requirements

- Java 17
- Maven

## Frontend used

### You can find the frontend of this project [here.](https://github.com/EduardoMtz1/ChatApp-Frontend)

## To run the application
```
    mvn spring-boot:run
```

## This application is running on
```
    http://localhost:9090
```
